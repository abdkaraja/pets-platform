# Plan 05-01: Domain Entities, Database Schema, DTOs, Validators, Service Layer Foundation

## Objective

Build the complete medical records and vet management foundation: two enums, six domain entities with factory methods, six EF Core configurations with proper indexes and FK relationships, all DTOs (read models, create DTOs, filter DTOs), four FluentValidation validators, two service interfaces (IVetService, IMedicalRecordService) with full implementations, and DI registration. After this plan the service layer is fully functional — all medical record CRUD, vet profile management, vet-pet assignment workflow, and the unified medical history timeline are wired up and tested via `dotnet build`.

**Purpose:** Establishes the data model and business logic that the UI layers (Plans 02 and 03) consume. All domain rules (vet assignment state machine, medical record creation with assignment verification, unified timeline query, admin approval) are enforced here.

**Output:** Working service layer with migration-ready schema, DI wired up, all validators registered.

Create `VetProfile.Create(dto.UserId, dto.FullName, dto.Clinic, dto.Specialty, dto.Bio, dto.ServicesOffered)`. Add to context. SaveChanges. Return mapped DTO.

  **UpdateProfileAsync(int id, UpdateVetProfileDto dto, string userId):** Validate DTO. Find profile by id. Verify `profile.UserId != userId` returns Failure (ownership check). Call `profile.UpdateDetails(...)`. Save. Return mapped DTO.

  **SearchVetsAsync(VetSearchFilterDto filter):** Query `_context.VetProfiles.Where(vp => vp.IsApproved)`. Apply optional filters: SearchTerm on FullName or Clinic (Contains, case-insensitive), Specialty (Contains), IsAvailable (exact match). Paginate. Return PagedResultDto<VetProfileDto>. Per D-02.

  **GetAcceptedAssignmentsAsync(string vetUserId):** Same query but Status == Accepted. Include VetProfile and Pet. Return mapped list.

  **GetActiveAssignmentAsync(int petId, string vetUserId):** Query for single assignment where PetId == petId && VetProfile.UserId == vetUserId && Status == Accepted. Return single or null.

  **GetAssignmentsForPetAsync(int petId):** Query all assignments for a pet (for admin view). Include VetProfile. Return mapped list.

  **ApproveVetAsync(int vetProfileId):** Find profile. Call `profile.Approve()`. Save. Return mapped DTO.

  **RejectVetAsync(int vetProfileId):** Find profile. Call `profile.Reject()`. Save. Return mapped DTO.

  **GetAvailabilityAsync(string vetUserId):** Query VetAvailability where VetProfile.UserId == vetUserId. Include VetProfile. Return mapped list.

  **UpdateAvailabilityAsync(string vetUserId, List<VetAvailabilityDto> schedule):** Find profile for userId. Remove existing availability entries. Add new ones from schedule using VetAvailability.Create. Save. Return success.
