# Plan 05-02: Vet Area — Controllers, Views, Dashboard, Profile, Medical Record Forms

## Objective

Build the complete Vet area (`Areas/Vet/`) with sidebar layout, dashboard with stat cards and assigned pets list, vet profile management (view/edit), assigned pets list with pet details showing medical records, pending assignment requests with accept/reject actions, and medical record creation forms for vaccinations (MED-01), medications (MED-02), and SOAP visit notes (MED-03). After this plan, a vet user can log in, see their dashboard, manage their profile, view assigned pets, accept/reject assignment requests, and create all three types of medical records for their assigned pets.

**Purpose:** Delivers the vet-facing experience — the primary data entry interface for the medical records module. Without this plan, there is no way to create medical records in the system.

**Output:** Full Vet area with 7 controllers, sidebar layout, dashboard view, profile views, assignment views, and 6 medical record views (3 create + 3 details).
