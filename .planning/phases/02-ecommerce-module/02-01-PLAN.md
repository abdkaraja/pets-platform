## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → API (forms/AJAX) | User input crosses from browser to server — all form submissions, AJAX calls |
| Server → Stripe API | Payment session creation — secret key used server-side only |
| Stripe → Server (webhook) | Webhook events must be signature-validated |
| Client → Stripe (Payment Element) | Card data handled by Stripe.js — never touches our server |
| User context (UserId) | Cart/order operations must be scoped to authenticated user |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-2-01 | Tampering | Stripe webhook endpoint | critical | mitigate | Validate Stripe-Signature header using EventUtility.ConstructEvent() — reject invalid signatures |
| T-2-02 | Tampering | Cart/checkout price manipulation | high | mitigate | Server computes all prices from DB; LockedPrice at add time; OrderItem.UnitPrice copied from LockedPrice at checkout |
| T-2-03 | Tampering | Stock overselling race condition | high | mitigate | Re-check stock at checkout time in CreateOrderFromCartAsync; ReduceStock throws on insufficient |
| T-2-04 | Elevation of Privilege | Unauthorized order status change | high | mitigate | Policy-based authorization: only Admin role with Orders.Manage claim can call UpdateOrderStatusAsync |
| T-2-05 | Information Disclosure | Cart/order data leakage across users | medium | mitigate | All cart/order queries filter by UserId from User.Identity; ownership verified before mutation |
| T-2-06 | Tampering | CSRF on checkout/payment forms | medium | mitigate | Anti-forgery tokens on all POST forms (ASP.NET Core default) |
| T-2-07 | Information Disclosure | Stripe secret key in client code | critical | mitigate | Secret key only in appsettings; StripeClient server-side only; only PublishableKey sent to client |
| T-2-SC | Tampering | Stripe.net NuGet package supply chain | high | mitigate | Package verified as official Stripe SDK (github.com/stripe/stripe-dotnet, 14+ years, maintained by Stripe employees) |
