# Requirements Archive: Pet Platform v1.0

**Archived:** 2026-07-21
**Status:** ✅ SHIPPED

## v1 Requirements

### Authentication

- [x] **AUTH-01**: User can sign up with email and password — Phase 1
- [x] **AUTH-02**: User receives email verification after signup — Phase 1
- [x] **AUTH-03**: User can reset password via email link — Phase 1
- [x] **AUTH-04**: User session persists across browser refresh — Phase 1
- [x] **AUTH-05**: Account lockout after failed login attempts — Phase 1
- [x] **AUTH-06**: System supports 4 roles: Admin, Customer, Vet, ServiceProvider — Phase 1
- [x] **AUTH-07**: Claims-based Permissions layered on Identity roles — Phase 1
- [x] **AUTH-08**: Policy-based Authorization in MVC controllers/actions — Phase 1

### Pet Management

- [x] **PET-01**: User can create pet profile with name, species, breed, age, weight, photo — Phase 1
- [x] **PET-02**: User can view/edit/delete own pet profiles — Phase 1
- [x] **PET-03**: User can view other users' pet profiles (public view) — Phase 1
- [x] **PET-04**: Multi-pet support per owner — Phase 1
- [x] **PET-05**: Customer profile with address, phone, city, notification preferences — Phase 1
- [x] **PET-06**: My Account page for viewing/editing customer data — Phase 1

### E-Commerce

- [x] **ECOM-01**: User can browse product catalog with search/filter — Phase 2
- [x] **ECOM-02**: User can filter products by pet type, category, price, brand — Phase 2
- [x] **ECOM-03**: User can add products to shopping cart — Phase 2
- [x] **ECOM-04**: User can checkout with payment — Phase 2
- [x] **ECOM-05**: User can view order history and status — Phase 2
- [x] **ECOM-06**: Admin can manage products (CRUD) — Phase 2
- [x] **ECOM-07**: Admin can manage categories — Phase 2
- [x] **ECOM-08**: Admin can manage inventory — Phase 2

### Adoption

- [x] **ADPT-01**: User can browse adoptable pet listings — Phase 3
- [x] **ADPT-02**: User can search/filter adoptable pets by species, breed, age, size, location — Phase 3
- [x] **ADPT-03**: User can submit adoption application form — Phase 3
- [x] **ADPT-04**: Shelter can review/approve/reject adoption applications — Phase 3
- [x] **ADPT-05**: Adopter receives application status updates — Phase 3

### Lost Pets

- [x] **LOST-01**: User can report lost/found pets with details — Phase 4
- [x] **LOST-02**: User can search lost pets by species, breed, color, location, date — Phase 4
- [x] **LOST-03**: User receives basic alerts for matching lost pets — Phase 4

### Medical Records

- [x] **MED-01**: Vet can create vaccination history records — Phase 5
- [x] **MED-02**: Vet can create medication records — Phase 5
- [x] **MED-03**: Vet can create visit notes — Phase 5
- [x] **MED-04**: Pet owner can view pet's medical records — Phase 5

### Admin Dashboard

- [x] **ADMN-01**: Admin can view all users — Phase 1
- [x] **ADMN-02**: Admin can activate/deactivate user accounts — Phase 1
- [x] **ADMN-03**: Admin can assign roles to users — Phase 1
- [x] **ADMN-04**: Admin can create new roles — Phase 1
- [x] **ADMN-05**: Admin can assign permissions to roles — Phase 1
